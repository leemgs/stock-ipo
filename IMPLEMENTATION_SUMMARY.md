# IPO 단타 분석 프로그램 구현 요약

## 구현 완료 내역

### 1. 종목 선정 (✓ 완료)
- 2026년 1~2월 공모주 데이터 구조 설계
- 예상 수익률 순으로 자동 정렬
- 각 종목별 장점/단점 요약 포함
- 5개 샘플 종목 제공 (적합 3개, 부적합 2개)

### 2. 단타 적합성 판단 (✓ 완료)
목표: 상장일 +5~30% 수익

4가지 부적합 기준 구현:
1. ✓ 의무보유확약 < 10%
2. ✓ 유통가능물량 > 35%
3. ✓ 공모가 밴드 최상단 초과
4. ✓ 유사 업종 상장일 평균 수익률 < 5%

**추가 기능:**
- 적합 종목에 대한 주의사항(warnings) 표시
- 상세한 판단 근거 제공

### 3. 매도 타이밍 (✓ 완료)
각 종목별 제공:
- ✓ 가장 불리한 구간 (시간대 + 사유)
- ✓ 가장 안전한 매도 타이밍 (시간대 + 사유)
- ✓ 수급 구조 기반 추천 근거

수급 강도에 따른 3단계 전략:
1. 강한 수급: 09:30~10:30 초반 고점 매도
2. 중간 수급: 09:15~09:45 시초가 직후 매도
3. 약한 수급: 09:00~09:20 조기 매도

- 업종 수익률 미달: 1건 ✓

## 확장성

사용자는 `create_sample_ipo_data()` 함수를 수정하거나 직접 IPOStock 객체를 생성하여 실제 공모주 데이터를 분석할 수 있습니다.

```python
from datetime import date
from ipo_analyzer import IPOStock, IPOAnalyzer

# 실제 데이터로 교체
my_stocks = [
    IPOStock(
        name="실제종목명",
        listing_date=date(2026, 1, 15),
        # ... 실제 데이터
    )
]

analyzer = IPOAnalyzer(my_stocks)
report = analyzer.generate_report()
print(report)
```
