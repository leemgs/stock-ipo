# 공모주 IPO 단타 전략 자동화 가이드

이 문서는 공모주(IPO) 청약 이후 ‘상장일 당일 매도’만을 목표로 하는 단타 전략을 체계적으로 판단하고 실행하기 위한 가이드이자 프롬프트입니다.
장기 보유, 기업 가치 분석, 중장기 성장성은 완전히 배제하고, 오직 상장일 기준 +5~30% 수익이 발생할 확률이 있는지 여부만을 판단 대상으로 삼습니다.
본 가이드는 과거 공모주 데이터와 **수급 구조(의무보유확약, 유통가능물량, 공모가 위치, 업종별 상장일 성과)**를 핵심 기준으로 하여,

* 월별 단타 유망 공모주 선별
* 단타 적합/부적합의 기계적인 판정
* 상장일 아침 기준 최적의 매도 타이밍 제시
  
까지를 단계적으로 수행하도록 설계되었습니다.

감정·기대·희망 회로를 배제하고, 개인 투자자가 가장 불리해지는 구간을 피하며 생존 확률을 높이는 것을 최우선 목표로 합니다.
