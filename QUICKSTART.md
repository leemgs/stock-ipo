# 빠른 시작 가이드

## 웹 애플리케이션 사용 (추천!)

컴퓨터 지식이 없어도 쉽게 사용할 수 있습니다.

### 1단계: 프로그램 다운로드

```bash
git clone https://github.com/leemgs/stock-ipo.git
cd stock-ipo
```

### 2단계: 필요한 프로그램 설치

```bash
pip install -r requirements.txt
```

### 3단계: 웹 서버 실행

```bash
python3 web_app.py
```

다음과 같은 메시지가 나타나면 성공입니다:
```
================================================================================
공모주 IPO 단타 분석 웹 애플리케이션
================================================================================

웹 브라우저에서 다음 주소로 접속하세요:
  http://localhost:5000
```

### 4단계: 웹 브라우저에서 접속

웹 브라우저 (크롬, 사파리, 엣지 등)를 열고 주소창에 입력:
```
http://localhost:5000
```

### 5단계: 분석 실행

화면에 보이는 **"📊 샘플 데이터로 분석 시작"** 버튼을 클릭하면 끝!

---

## 명령줄 사용 (개발자용)

터미널에서 직접 실행하는 방법:

```bash
python3 ipo_analyzer.py
```

결과가 화면에 출력되고 `ipo_analysis_report.txt` 파일로 저장됩니다.
