# 📈 공모주 청약 단타 전략 가이드

## 1. 개요

이 저장소는 **공모주(IPO) 청약 후 상장일 당일 매도(단타)** 전략을 중심으로,  
개인 투자자가 수급 구조를 기반으로 **확률적으로 유리한 선택**을 하기 위한 가이드를 제공합니다.

본 전략은 장기 투자, 기업 가치 분석, 재무제표 평가는 **완전히 배제**하며  
오직 **상장일 하루 동안 발생하는 수급과 가격 변동성**만을 다룹니다.

---

## 2. 공모주 청약이란?

**공모주 청약(IPO Subscription)**이란  
비상장 기업이 증권시장에 상장하면서 일반 투자자에게 주식을 배정하는 과정입니다.

- 정해진 기간에 증권사를 통해 청약
- 상장일에 주식이 계좌에 입고
- 상장 이후 매도 가능

---

## 3. 공모주 단타 전략이란?

공모주 단타는  
**상장일 당일(IPO Day)에 매도하는 것을 전제로 한 초단기 투자 전략**입니다.

- 목표: 상장일 기준 **+5~30% 수익**
- 보유 기간: 1일 (상장일)
- 핵심 요소:
  - 의무보유확약 비율
  - 상장 직후 유통가능 물량
  - 공모가 위치(밴드 내/초과)
  - 유사 업종의 최근 상장일 성과

---

## 4. 공모주 단타의 장점

### ✅ 장점

- **짧은 자금 묶임**
  - 청약 → 상장 → 매도까지 기간이 명확
- **시장 리스크 노출 최소화**
  - 장기 변동성, 악재 노출 회피
- **데이터 기반 판단 가능**
  - 수급 구조는 수치로 명확히 판단 가능
- **개인 투자자도 참여 가능**
  - 대형 자금 없이도 전략 수행 가능

---

## 5. 공모주 단타의 단점

### ❌ 단점

- **상장일 변동성 매우 큼**
  - 시초가 이후 급등락 빈번
- **조건이 나쁘면 즉시 실패**
  - 확약 부족, 유통물량 과다 시 손실 확률 급증
- **감정 개입 시 실패 확률 상승**
  - “더 갈 것 같다”는 기대가 손실로 이어지기 쉬움
- **모든 공모주가 대상은 아님**
  - 대부분의 공모주는 단타에 부적합

---

## 6. 단타 부적합 판정 기준 (중요)

아래 중 **하나라도 해당되면 단타 부적합**으로 판단합니다.

- 의무보유확약 비율 < 10%
- 상장 직후 유통가능 물량 > 35%
- 공모가가 희망 밴드 최상단 초과 확정
- 최근 유사 업종 상장일 평균 수익률 < 5%

---

## 7. 프로젝트 목표

- 공모주 중 **단타 가능한 종목만 선별**
- 상장일 기준 **개인이 가장 불리한 구간 회피**
- 가장 **안전한 매도 타이밍 제시**
- 감정이 아닌 **확률과 구조 중심 의사결정**

---
