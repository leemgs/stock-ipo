# stock-ipo

공모주 청약 단타 수익 분석 프로그램

## 개요

이 프로그램은 공모주 IPO 단기 수급과 상장일 매도 확률을 과거 데이터와 수급 구조로 판단하여 상장일 당일 단타 수익을 노리는 투자자를 위한 분석 도구입니다.

## 주요 기능

1. **종목 선정**: 2026년 1~2월 공모주 중 청약 후 당일 매도 시 기대수익이 높은 종목을 선정하고 장단점을 요약
2. **단타 적합성 판단**: 상장일 기준 +5~30% 수익 목표로 4가지 기준으로 평가
   - 의무보유확약 <10% → 부적합
   - 유통가능물량 >35% → 부적합
   - 공모가 밴드 최상단 초과 → 부적합
   - 유사 업종 상장일 평균 수익률 <5% → 부적합
3. **매도 타이밍**: 상장일 아침 기준, 종목별로 가장 불리한 구간과 가장 안전한 매도 타이밍 제시

#### 직접 코드에서 사용

```python
from datetime import date
from ipo_analyzer import IPOStock, IPOAnalyzer

# 공모주 데이터 생성
stocks = [
    IPOStock(
        name="종목명",
        listing_date=date(2026, 1, 15),
        # ... 기타 필드
    )
]

# 분석 실행
analyzer = IPOAnalyzer(stocks)
report = analyzer.generate_report()
print(report)
```

### 출력

프로그램은 다음 두 가지 방식으로 결과를 출력합니다:
1. 콘솔 화면에 분석 보고서 출력
2. `ipo_analysis_report.txt` 파일로 보고서 저장

## 분석 보고서 구조

### 1. 종목 선정
- 단타 적합 종목 목록 (예상 수익률 순)
- 각 종목의 기본 정보, 장점, 단점

### 2. 단타 적합성 판단
- 각 종목별 적합/부적합 판정
- 판정 근거 상세 설명

### 3. 매도 타이밍
- 종목별 가장 불리한 시간대
- 종목별 가장 안전한 매도 타이밍
- 타이밍 추천 근거

```
    ...

```

## 커스터마이징

`create_sample_ipo_data()` 함수에서 실제 공모주 데이터를 추가하거나 수정할 수 있습니다.

```python
IPOStock(
    name="종목명",
    listing_date=date(2026, 1, 15),
    ipo_price=25000,
    price_band_min=23000,
    price_band_max=27000,
    mandatory_holding_pct=15.5,
    available_float_pct=28.3,
    sector="업종",
    sector_avg_return_pct=12.8,
    expected_return_pct=18.5,
    strengths=["장점1", "장점2"],
    weaknesses=["단점1", "단점2"]
)
```

## 면책 조항

본 프로그램의 분석은 과거 데이터와 수급 구조 기반 판단이며, 실제 시장 상황에 따라 달라질 수 있습니다. 투자 결정은 본인의 책임 하에 이루어져야 합니다.
