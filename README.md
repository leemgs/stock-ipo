
# 📈 공모주 청약 단타 분석 프로그램

## 🎯 개요

이 저장소는 **공모주(IPO) 청약 후 상장일 당일 매도(단타)** 전략을 지원하는 자동화 분석 도구를 제공합니다.

**핵심 기능:**
- 🤖 공모주 적합성 자동 평가 (4가지 기준)
- 📊 수급 구조 기반 매도 타이밍 추천
- 🌐 웹 애플리케이션으로 누구나 쉽게 사용
- 📈 과거 데이터 기반 확률적 판단 지원

본 전략은 장기 투자, 기업 가치 분석은 **완전히 배제**하며, 오직 **상장일 하루 동안의 수급과 가격 변동성**만을 다룹니다.

## 📚 공모주 단타 전략 이해하기

### 공모주 청약이란?

**공모주 청약(IPO Subscription)**이란 비상장 기업이 증권시장에 상장하면서 일반 투자자에게 주식을 배정하는 과정입니다.

- 정해진 기간에 증권사를 통해 청약
- 상장일에 주식이 계좌에 입고
- 상장 이후 매도 가능

### 공모주 단타 전략이란?

**상장일 당일(IPO Day)에 매도하는 것을 전제로 한 초단기 투자 전략**입니다.

- **목표**: 상장일 기준 **+5~30% 수익**
- **보유 기간**: 1일 (상장일)
- **핵심 요소**:
  - 의무보유확약 비율
  - 상장 직후 유통가능 물량
  - 공모가 위치(밴드 내/초과)
  - 유사 업종의 최근 상장일 성과

### ✅ 장점

- **짧은 자금 묶임**: 청약 → 상장 → 매도까지 기간이 명확
- **시장 리스크 노출 최소화**: 장기 변동성, 악재 노출 회피
- **데이터 기반 판단 가능**: 수급 구조는 수치로 명확히 판단 가능
- **개인 투자자도 참여 가능**: 대형 자금 없이도 전략 수행 가능

### ❌ 단점

- **상장일 변동성 매우 큼**: 시초가 이후 급등락 빈번
- **조건이 나쁘면 즉시 실패**: 확약 부족, 유통물량 과다 시 손실 확률 급증
- **감정 개입 시 실패 확률 상승**: "더 갈 것 같다"는 기대가 손실로 이어지기 쉬움
- **모든 공모주가 대상은 아님**: 대부분의 공모주는 단타에 부적합

## 🔍 자동 분석 기능

이 프로그램은 공모주를 자동으로 분석하여 단타 적합성을 판단합니다.

### 단타 부적합 판정 기준

아래 중 **하나라도 해당되면 자동으로 단타 부적합**으로 판단합니다:

1. ❌ 의무보유확약 비율 < 10%
2. ❌ 상장 직후 유통가능 물량 > 35%
3. ❌ 공모가가 희망 밴드 최상단 초과
4. ❌ 최근 유사 업종 상장일 평균 수익률 < 5%

### 매도 타이밍 자동 추천

수급 강도를 계산하여 3가지 전략 중 하나를 추천합니다:

- **강한 수급** (확약 높음, 물량 적음): 09:30~10:30 초반 고점 매도
- **중간 수급**: 09:15~09:45 시초가 직후 빠른 매도
- **약한 수급**: 09:00~09:20 즉시 매도 또는 손절 고려

## 💻 사용 방법

### CLI (커맨드 라인) - 개발자용

#### 기본 실행

```bash
python3 ipo_analyzer.py
```

결과가 화면에 출력되고 `ipo_analysis_report.txt` 파일로 저장됩니다.

#### 커스텀 데이터로 분석

```python
from datetime import date
from ipo_analyzer import IPOStock, IPOAnalyzer

# 공모주 데이터 생성
stocks = [
    IPOStock(
        name="종목명",
        listing_date=date(2026, 1, 15),
        ipo_price=25000,
        price_band_min=23000,
        price_band_max=27000,
        mandatory_holding_pct=15.5,
        available_float_pct=28.3,
        sector="업종",
        sector_avg_return_pct=12.8,
        expected_return_pct=18.5,
        strengths=["장점1", "장점2"],
        weaknesses=["단점1", "단점2"]
    )
]

# 분석 실행
analyzer = IPOAnalyzer(stocks)
report = analyzer.generate_report()
print(report)
```

## 🚀 배포 및 운영

### Ubuntu 서버 자동 실행

```bash
# systemd 서비스로 등록
sudo cp stock-ipo.service /etc/systemd/system/
sudo systemctl enable stock-ipo.service
sudo systemctl start stock-ipo.service
```

포트 5701로 서비스가 자동 시작됩니다. 상세한 내용은 [DEPLOYMENT.md](DEPLOYMENT.md)를 참고하세요.
